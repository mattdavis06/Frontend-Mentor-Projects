# 01- Frontend Mentor - Interactive Pricing Component

This is a solution to the [Interactive pricing component on Frontend Mentor](https://www.frontendmentor.io/challenges/tip-calculator-app-ugJNGbJUX). Frontend Mentor challenges help you improve your coding skills by building realistic projects.

### The challenge

![Design preview for the Tip calculator app coding challenge](./design/desktop-preview.jpg)

Users should be able to:

- View the optimal layout for the app depending on their device's screen size
- See hover states for all interactive elements on the page
- Calculate the correct tip and total cost of the bill per person
